# Software Developer Portfolio

This repository contains the source code for my personal software developer portfolio, built with HTML5, CSS3, and JavaScript and deployed using GitHub Pages.

The portfolio presents a concise overview of my background, technical skills, and selected academic and personal projects completed during and after my BSc (Hons) in Software Development at South East Technological University (SETU), Carlow.

## Portfolio Overview

The website includes:

* A short professional profile
* Technical skills overview
* Selected project cards with links to GitHub repositories and live demos where available
* Contact form
* Link to a general CV

## Featured Projects

### Smart Marathon Runner

Final-year Android project designed to help novice marathon runners analyse Strava activities, monitor training progress, and receive adaptive feedback based on pace execution, consistency, and plan completion.

The project received the Netwatch Software Development Award at the SETU Carlow Showcase 2026.

Technologies include Kotlin, Jetpack Compose, MVVM, Room, Retrofit, OAuth 2.0, and the Strava API.

### Barbershop System & Cloud Migration

Full-stack barbershop booking system with product sales, Stripe payments, role-based dashboards, Docker containerisation, and AWS cloud migration from a shared hosting environment.

Technologies include PHP, MySQL, MVC, Stripe API, Docker, AWS, and cloud deployment concepts.

### Marathon Finish Time Predictor

Data science project using a real-world marathon dataset to clean, explore, train, and evaluate a linear regression model, deployed as an interactive Streamlit prediction app.

Technologies include Python, Pandas, Scikit-learn, Linear Regression, EDA, and Streamlit.
